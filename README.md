## Quick Installation
  ```ruby
    https://github.com/robiul-islam-rubel/GYM.git
    cd GYM
   ``` 
### Composer
```ruby
    composer update
```  
 ### For Migration table in database [Create database name as ```gym```]
 ```ruby
    php artisan migrate
 ```  
### Server ON ```url: http://127.0.0.1:8000/```
```ruby
    php artisan serve
```
